## Основные изменения

- единый SG Window Standard для дополнительных окон;
- правильные размеры, скругления, заголовки и нижние панели;
- отсутствие белых системных полос в дополнительных окнах;
- окна настроек остаются открытыми после кнопки «Применить»;
- проверка фактического состояния подключения;
- сравнение параметров DPI до и после применения;
- пользовательская проверка обновлений только для Xray и sing-box;
- Mihomo исключён из пользовательских уведомлений;
- AmneziaWG обновляется вместе со всем SG Client;
- самостоятельное обновление SG Client отключено.

## Поддерживаемые профили

- VLESS REALITY;
- VLESS XHTTP + REALITY;
- VLESS XHTTP + TLS;
- Hysteria2 + QUIC + TLS;
- AmneziaWG.

## Первый запуск

1. Полностью распакуйте ZIP.
2. Запустите `SG-Client-040.exe`.
3. Импортируйте один профиль.
4. Подключитесь.
5. Дождитесь состояния «Подключено».
6. Проверьте интернет, внешний IP и журнал.

Подробное руководство: [docs/START-HERE.md](docs/START-HERE.md).
