# TUN и маршрутизация

## TUN

SG Client запускает только один активный TUN-движок. При переключении профиля текущее подключение сначала останавливается, затем запускается выбранный профиль.

## Маршрутизация

Для категорий доступны действия:

- `Direct` — напрямую;
- `VPN` — через выбранный профиль;
- `Block` — заблокировать.

Пользовательские правила должны проверяться по фактически созданному `config.json` и реальным сетевым тестом.

## Раздельный TUN

Для Xray и sing-box можно задавать приложения, IP-адреса и подсети вне TUN. Для AmneziaWG применяются адреса и подсети; исключение отдельных приложений зависит от системного фильтра Windows.
