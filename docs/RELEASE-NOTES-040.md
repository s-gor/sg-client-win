# SG Client 040

Первый подготовленный portable-релиз единого Windows-клиента для SG-Panel и SG-AWG-Panel.

## Основное

- единый интерфейс для Xray, sing-box и AmneziaWG;
- управление TUN и переключением профилей;
- единый стандарт дополнительных окон;
- проверка фактического состояния подключения;
- сравнение параметров DPI до и после применения;
- умная маршрутизация, DNS, локальная сеть и раздельный TUN;
- обновления только Xray и sing-box;
- Mihomo исключён из пользовательских уведомлений.

## Установка

1. Скачайте `SG-CLIENT-040-PORTABLE-x64.zip`.
2. Полностью распакуйте архив.
3. Запустите `SG-Client-040.exe`.

## Требования

- Windows 10/11 x64;
- права администратора для TUN.

Контрольные суммы находятся в `SHA256SUMS.txt`.
