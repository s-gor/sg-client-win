# Проверка подключения

## Признаки нормальной работы

1. Статус показывает «Подключено».
2. Кнопка предлагает отключить текущий режим.
3. Выбранный профиль активен.
4. Счётчики трафика меняются.
5. Сайты открываются.
6. Внешний IP соответствует ожидаемому маршруту.
7. В журнале нет повторяющихся ошибок ядра.

## Проверка TUN

Проверьте программу, которая не умеет использовать системный прокси. После отключения TUN обычная сеть должна восстановиться.

## Проверка системного прокси

Проверьте браузер и настройки прокси Windows. Помните: не все программы используют системный прокси.

## Если не работает

1. Верните маршрутизацию к «Весь интернет через VPN».
2. Отключите пользовательский DNS.
3. Верните DPI в автоматический режим.
4. Отключите Kill Switch.
5. Повторите подключение и изучите журнал.
