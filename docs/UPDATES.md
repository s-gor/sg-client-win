# Обновление Xray, sing-box и GeoFiles

## Xray и sing-box

Доступны стабильная, предварительная, точная версия и локальный файл.

Перед заменой SG Client:

1. загружает кандидат;
2. проверяет SHA-256;
3. проверяет запуск;
4. тестирует подходящий текущий конфиг;
5. создаёт резервную копию;
6. заменяет файлы;
7. выполняет повторную проверку;
8. автоматически откатывает при ошибке.

Если подходящего профиля sing-box нет, возможна ограниченная экспертная проверка только ядра с явным предупреждением.

Понижение версии не должно предлагаться как обычное обновление.
