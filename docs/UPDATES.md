# Обновления компонентов

## Что проверяет SG Client 040

Пользовательская проверка обновлений выполняется только для:

- Xray;
- sing-box.

Mihomo исключён из проверки и пользовательских уведомлений.

## AmneziaWG

AmneziaWG не обновляется отдельно из интерфейса. Новая версия поставляется вместе с новым выпуском SG Client, чтобы клиент, AWG-компоненты и логика интеграции оставались согласованными.

## Сам SG Client

Самостоятельное обновление SG Client отключено. Новый выпуск скачивается со страницы Releases и распаковывается в новую отдельную папку.

## Безопасный порядок обновления

1. Отключите активный профиль.
2. Закройте SG Client.
3. Скачайте новый Portable Release.
4. Распакуйте его в новую папку.
5. Не копируйте старые журналы и рабочие конфигурации поверх новой версии без необходимости.
6. Запустите новый клиент и проверьте один профиль.
7. Старую папку удаляйте только после успешной проверки.
