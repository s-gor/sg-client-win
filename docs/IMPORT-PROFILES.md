# Импорт профилей

## Из SG-Panel

Для первого подключения используйте прямую ссылку конкретного профиля.

1. Откройте `Clients`.
2. Выберите клиента.
3. Откройте ссылки и подписку.
4. Скопируйте нужную активную ссылку.
5. В SG Client откройте добавление профиля.
6. Выберите импорт из буфера обмена.
7. Сохраните найденный профиль.
8. Выберите его и подключитесь.

Если SG-Panel показывает несколько Inbound, импортируйте ссылки отдельно. Имена `Primary`, `Backup` и `Alt` помогают понять, к какой точке относится профиль.

Неактивная сохранённая ссылка может быть скопирована, но она не заработает, пока соответствующий Inbound не активен на сервере.

## Из SG-AWG-Panel

1. Откройте страницу клиента.
2. Скопируйте полный текст конфигурации AmneziaWG либо скачайте `.conf`.
3. В SG Client выберите импорт AmneziaWG.
4. Вставьте текст или выберите файл.
5. Сохраните профиль.
6. Подключитесь.

Не удаляйте строки маскировки из конфигурации. Они должны импортироваться вместе с адресом, ключами, DNS, MTU и endpoint.

## Какой вариант выбрать

| Данные панели | Действие в SG Client |
|---|---|
| `vless://...` | импорт ссылки из буфера |
| `hysteria2://...` | импорт ссылки из буфера |
| полный текст `[Interface]` / `[Peer]` | импорт AmneziaWG из буфера |
| файл `.conf` | импорт AmneziaWG из файла |
| подписка | добавлять после проверки прямой ссылки |

## После импорта

Проверьте:

- понятное имя профиля;
- правильный тип;
- адрес сервера и порт;
- отсутствие явного сообщения об ошибке;
- возможность выбрать профиль в основном окне.

Секретные параметры не публикуйте в Issues и открытых сообщениях.
