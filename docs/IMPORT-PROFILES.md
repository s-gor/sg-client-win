# Импорт профилей и подписок

## Поддерживаемые варианты

- VLESS-ссылка;
- Hysteria2-ссылка;
- AmneziaWG `.conf` или полный текст;
- совместимая подписка;
- профиль SG-Panel / SG-AWG-Panel.

## Прямая ссылка

Для первого теста предпочтительна прямая ссылка. Импортируйте её, сохраните профиль и подключитесь без дополнительных изменений.

## Подписка

Укажите название и URL. Обновление можно выполнять напрямую или через активное подключение. При ошибке сохраняется последний рабочий список.

## Серверно-зависимые параметры

Поля REALITY, XHTTP, TLS, Flow/Vision, auth и AmneziaWG должны приходить из профиля. Не конструируйте их наугад.

## Локальная копия

Профиль подписки или SG-профиль лучше не редактировать напрямую. Для экспериментов создайте независимую локальную копию.
