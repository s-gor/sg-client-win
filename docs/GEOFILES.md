# GeoFiles и источники

## Что это

- `geoip.dat` содержит группы IP-адресов и сетей;
- `geosite.dat` содержит доменные категории;
- SRS-файлы используются правилами sing-box.

GeoFiles влияют на умную маршрутизацию: какие ресурсы идут напрямую, через VPN или блокируются.

## Источники

### Комплект SG Client — рекомендуется

Проверенный набор, включённый в Portable. Работает без загрузки и может быть восстановлен кнопкой возврата к комплектным файлам.

### Loyalsoldier

Внешний регулярно обновляемый набор GeoIP/GeoSite для Xray. SG Client проверяет файлы, но не управляет содержимым источника.

### RunetFreedom

Внешний набор, ориентированный на российский сегмент и специализированные категории маршрутизации.

### Пользовательские URL

Отдельные URL для `geoip.dat` и `geosite.dat`. Используйте только доверенный источник администратора.

### Локальные файлы

Позволяют установить заранее скачанный или собственный набор.

## Что показывает SG Client

- текущий источник;
- размер;
- дату;
- SHA-256;
- статус проверки.

## Безопасная замена

1. Файлы загружаются во временную папку.
2. Считается SHA-256.
3. Выполняется проверка установленным Xray.
4. Текущий набор сохраняется в `guiBackups\geofiles`.
5. Новые файлы устанавливаются.
6. При ошибке выполняется автоматический откат.

## После замены

Проверьте российские ресурсы, заблокированные сайты, рекламу и пользовательские категории. При неожиданном поведении верните комплект SG Client или восстановите резервную копию.
