# Быстрый старт

1. Скачайте portable ZIP из GitHub Releases.
2. Полностью распакуйте архив.
3. Запустите `SG-Client-040.exe`.
4. Импортируйте ссылку SG-Panel или конфигурацию SG-AWG-Panel.
5. Выберите профиль.
6. Нажмите «Включить TUN».
7. Дождитесь зелёного состояния «TUN работает».

Для отключения используйте кнопку SG Client, а не завершение процесса через Диспетчер задач.

При первом запуске Windows может запросить разрешение администратора для TUN.
