# С чего начать

Документ относится к SG Client `v0.0.60`.

## Новая установка

1. Скачайте `SG-CLIENT-060-PORTABLE-x64.zip` со страницы релиза.
2. Создайте отдельную папку, например `D:\Programs\SG-Client-060`.
3. Полностью распакуйте туда архив.
4. Запустите `SG-Client-060.exe`.
5. Разрешите запуск с правами администратора.
6. Импортируйте один профиль или добавьте подписку.
7. Выберите профиль.
8. Включите TUN или системный прокси.
9. Проверьте интернет, внешний IP, трафик и журнал.

## Что импортировать

- из SG-Panel — VLESS/Hysteria2-ссылку или подписку;
- из SG-AWG-Panel — полный текст AmneziaWG либо файл `.conf`;
- из другого источника — совместимую стандартную ссылку или конфигурацию.

## Рекомендуемый порядок

```text
получить ссылку или конфигурацию
→ импортировать профиль
→ выбрать профиль
→ включить TUN
→ проверить интернет
→ только затем менять DNS, маршрутизацию, DPI или Kill Switch
```

## Три режима

- **TUN** — основной рекомендуемый режим для всего системного трафика;
- **Системный прокси** — для приложений, использующих настройки прокси Windows;
- **Локальный прокси** — для ручной настройки отдельных программ.
