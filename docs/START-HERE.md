# С чего начать

Документ относится к SG Client `v0.0.40`.

## Новая установка

1. [Скачайте SG Client 040 Portable x64](https://github.com/s-gor/sg-client-win/releases/download/v0.0.40/SG-CLIENT-040-PORTABLE-x64.zip).
2. Создайте отдельную папку, например `D:\Programs\SG-Client-040`.
3. Полностью распакуйте туда архив.
4. Запустите `SG-Client-040.exe`.
5. Разрешите запуск с правами администратора.
6. Импортируйте один профиль.
7. Подключитесь и проверьте реальный доступ в интернет.
8. После проверки можно настраивать DNS, маршрутизацию, DPI и Kill Switch.

## Что импортировать

Из SG-Panel копируется прямая VLESS или Hysteria2-ссылка. Из SG-AWG-Panel копируется полный текст AmneziaWG-конфигурации либо скачивается `.conf`.

Подробности: [Импорт профилей](IMPORT-PROFILES.md).

## Основной порядок работы

```text
Получить ссылку или конфигурацию
→ импортировать профиль
→ выбрать профиль
→ подключить
→ дождаться состояния «Подключено»
→ проверить интернет и внешний IP
→ только затем менять дополнительные настройки
```

## Первый порядок проверки

1. Главная кнопка изменилась на отключение.
2. Состояние показывает, что подключение активно.
3. Счётчики трафика меняются.
4. Сайты открываются.
5. В журнале нет повторяющейся ошибки.
6. После отключения обычное соединение Windows восстанавливается.

Подробности: [Проверка подключения](CONNECTION-CHECK.md).
