# Маскировка DPI

## Режимы

SG Client предусматривает следующие пользовательские варианты:

- автоматически — рекомендуется;
- выключена;
- дробление TLS/ClientHello;
- дробление TLS + шум — экспериментально.

## Когда использовать

Начинайте с автоматического режима. Если соединение работает стабильно, дополнительное вмешательство не требуется.

Отключённый режим полезен для сравнения и диагностики. Дробление стоит пробовать, когда поддерживаемый Xray-профиль не устанавливает стабильное TLS-соединение в конкретной сети.

Режим с шумом является экспериментальным. Он может помочь в одной сети и ухудшить работу в другой.

## Ограничения по протоколам

- VLESS/Xray: возможны управляемое дробление и шум;
- Hysteria2: обычное TLS-дробление Xray не применяется;
- AmneziaWG: маскировка задаётся параметрами самого AWG-профиля.

## Применение

Окно показывает выбранные параметры и фактически применяемый результат. После нажатия «Применить» окно остаётся открытым, чтобы можно было сравнить значения.

После изменения:

1. переподключите профиль;
2. проверьте несколько HTTPS-сайтов;
3. сравните скорость установления соединения;
4. посмотрите журнал;
5. при ухудшении вернитесь к автоматическому режиму.
