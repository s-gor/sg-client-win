# Маскировка DPI

## Назначение

Фрагментация может затруднить анализ TLS ClientHello некоторыми DPI-системами. Она не шифрует SNI и не гарантирует обход любого фильтра.

## Режимы

- автоматически — рекомендуется;
- выключено;
- дробление TLS/ClientHello;
- дробление TLS + шум — экспериментально;
- пользовательские параметры.

## Совместимость

- Xray/VLESS — управляемая фрагментация;
- Hysteria2 — обычное Xray-дробление не применяется;
- AmneziaWG — маскировка встроена в профиль.

## Ручной JSON

Последовательность:

```text
Редактировать → Проверить → тест ядром → Применить
```

Используйте только понятные параметры. При ухудшении связи нажмите «По умолчанию».
