# Проверка релиза

## Автоматически

- восстановление NuGet-зависимостей;
- запуск `ServiceLib.Tests`;
- Release-публикация WPF-проекта;
- проверка имени `SG-Client-040.exe`;
- проверка whitelist обновлений Xray и sing-box;
- проверка структуры portable ZIP;
- проверка отсутствия пользовательских профилей, конфигураций и журналов;
- SHA-256 всех release-файлов.

## Вручную

- чистый запуск portable на Windows 10/11 x64;
- импорт профиля;
- включение и отключение TUN;
- переключение профиля при работающем TUN;
- окна «Настройки», «Маршрутизация», «DPI», «Раздельный TUN», «Справка»;
- отсутствие белых системных полос;
- окно остаётся открытым после «Применить»;
- проверка Xray;
- проверка Hysteria 2;
- проверка AmneziaWG;
- проверка Kill Switch и аварийного сброса;
- в проверке обновлений отсутствует Mihomo.

Публикация разрешается только после прохождения обязательного smoke-теста.
