# Kill Switch и восстановление сети

Kill Switch предназначен для защиты TUN при аварийном разрыве. В системном или локальном прокси он не защищает весь компьютер.

## Перед включением

1. Убедитесь, что профиль стабильно работает.
2. Проверьте штатное отключение TUN.
3. Ознакомьтесь с журналом и встроенной справкой.
4. Создайте резервную копию настроек.

## Если интернет заблокирован

- снова запустите SG Client с правами администратора;
- отключите Kill Switch;
- выполните штатное отключение TUN;
- при необходимости перезагрузите Windows и проверьте сетевые правила.

Не публикуйте журналы без проверки: они могут содержать адреса и другие данные профиля.
