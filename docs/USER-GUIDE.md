# Руководство пользователя SG Client

Это руководство описывает функции SG Client `v0.0.40`.

## Назначение

SG Client запускает на Windows профили, созданные SG-Panel и SG-AWG-Panel. VLESS обслуживается Xray, Hysteria2 — sing-box, AmneziaWG — отдельным AWG-движком.

## Первый запуск

Portable не устанавливается в Windows. После распаковки запустите `SG-Client-040.exe` с правами администратора. Эти права нужны для TUN, системных маршрутов, Wintun и Kill Switch.

Программа создаёт рабочие папки рядом с собой только во время использования. Их нет в чистом релизе.

## Профили

### VLESS REALITY

Используется для обычного VLESS REALITY через TCP/RAW. Сертификат на клиенте не нужен. Импортируется прямой ссылкой из SG-Panel.

### VLESS XHTTP + REALITY

Использует XHTTP и REALITY. Path, mode и остальные параметры должны приходить из ссылки, созданной SG-Panel. Не переносите параметры вручную между разными профилями.

### VLESS XHTTP + TLS

Использует XHTTP через TLS. Домен, Path, SNI и режим XHTTP берутся из клиентской ссылки.

### Hysteria2

Работает через sing-box и QUIC/UDP. Для него важны UDP-доступность, TLS, auth и порт сервера. Обычное Xray-дробление TLS к Hysteria2 не применяется.

### AmneziaWG

Импортируется из полного текста конфигурации или файла `.conf`. Параметры `Jc`, `Jmin`, `Jmax`, `S1–S4` и `H1–H4`, если они присутствуют, являются частью профиля.

## Подключение

1. Выберите профиль.
2. Нажмите главную кнопку подключения.
3. Дождитесь окончания переходного состояния.
4. Проверьте состояние, трафик и интернет.
5. Для завершения нажмите отключение.

Не запускайте одновременно несколько копий SG Client из разных папок. Это может привести к конфликту процессов, TUN и рабочих конфигураций.

## TUN

TUN используется как системный сетевой режим. При его включении SG Client создаёт рабочую конфигурацию для выбранного движка, запускает процесс и применяет необходимые маршруты.

При нажатии «Выйти» клиент должен сначала отключить TUN и только затем завершить работу.

## DNS

Режим «DNS через TUN» направляет DNS-запросы в контур активного подключения. Если после его включения перестали открываться домены, сначала верните DNS к обычному режиму и повторно проверьте профиль без дополнительных настроек.

## Маршрутизация

Раздельный TUN позволяет направлять через активный профиль только выбранные направления. Системный режим направляет через профиль основной трафик Windows.

Правила должны соответствовать цели подключения. Не включайте одновременно несколько взаимоисключающих режимов только ради проверки.

## Маскировка DPI

Начинайте с автоматического режима. Если профиль работает без маскировки, не добавляйте экспериментальные параметры. Дробление TLS и шум применимы только к поддерживаемым Xray-сценариям.

Окно настройки показывает выбранные и применяемые параметры. После «Применить» окно остаётся открытым, чтобы можно было сравнить значения.

## Kill Switch

Kill Switch защищает от выхода трафика в обход активного подключения. При неправильной остановке процесса правила могут временно сохранить блокировку. Для аварийного снятия используется `RESET-KILL-SWITCH.cmd` из корня Portable.

## Значки и состояние

Зелёное состояние означает активное подключение. Жёлтое показывает выполняющуюся операцию. Серое или красное означает отключение либо ошибку.

Не оценивайте работу только по значку: проверяйте статус, трафик, интернет и журнал.

## Журнал

Журнал нужен для определения этапа ошибки: импорт, создание конфигурации, запуск движка, TUN, DNS, маршруты или Kill Switch.

Перед отправкой журнала проверьте, что в нём нет данных, которыми вы не хотите делиться. Пароли Hysteria2 и приватные ключи не следует публиковать.

## Обновления

В 040 пользовательская проверка обновлений относится только к Xray и sing-box. Mihomo не проверяется. AmneziaWG обновляется вместе с новым выпуском SG Client.

## Обычные сценарии

### Подключить ссылку SG-Panel

`Clients → выбрать клиента → скопировать прямую ссылку → SG Client → импорт из буфера → сохранить → подключить`.

### Подключить AmneziaWG

`SG-AWG-Panel → клиент → скопировать конфигурацию → SG Client → импорт AmneziaWG → сохранить → подключить`.

### Проверить новый профиль

Подключите профиль с настройками по умолчанию. Убедитесь, что открываются сайты. После этого по одному включайте DNS через TUN, раздельную маршрутизацию, DPI или Kill Switch.

### Сменить серверный профиль

Получите новую актуальную ссылку из панели. Не редактируйте старую ссылку вручную, если панель изменила transport, Path, порт, SNI, auth или параметры REALITY.

## После любого изменения

1. Переподключите профиль.
2. Дождитесь стабильного состояния.
3. Проверьте интернет.
4. Проверьте ожидаемый внешний IP.
5. Посмотрите последние строки журнала.
6. Убедитесь, что отключение возвращает обычную сеть.
