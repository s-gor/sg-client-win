# Portable: скачивание и запуск

## Официальный файл

[Скачать SG Client 040 Portable x64](https://github.com/s-gor/sg-client-win/releases/download/v0.0.40/SG-CLIENT-040-PORTABLE-x64.zip)

SHA-256 опубликован в [SHA256SUMS.txt](https://github.com/s-gor/sg-client-win/releases/download/v0.0.40/SHA256SUMS.txt).

## Распаковка

1. Создайте отдельную папку.
2. Извлеките туда всё содержимое ZIP.
3. Убедитесь, что рядом с `SG-Client-040.exe` находятся DLL, папка `bin` и `RESET-KILL-SWITCH.cmd`.
4. Запустите EXE из этой папки.

## Почему нет установщика

Основной формат SG Client — Portable ZIP. Установочный EXE не используется, чтобы не добавлять лишний этап установки и дополнительные предупреждения установщика.

Неподписанная тестовая сборка всё равно может вызвать предупреждение Windows SmartScreen о неизвестном издателе. Проверяйте, что файл скачан с официальной страницы релиза.

## Что создаётся при работе

Программа может создать рабочие каталоги:

- `guiConfigs`;
- `guiLogs`;
- `guiTemps`;
- `binConfigs`.

В чистом релизе их нет. Они относятся к конкретному компьютеру и не должны переноситься в публичный архив.

## Перенос на другой компьютер

Перенесите чистый Portable и заново импортируйте профили из панелей. Не публикуйте рабочие папки и личные конфигурации.
