# Portable-версия SG Client 060

## Установка не требуется

Полностью распакуйте `SG-CLIENT-060-PORTABLE-x64.zip` в отдельную папку и запустите `SG-Client-060.exe`.

Не запускайте EXE непосредственно из ZIP: ядрам, настройкам, журналам, резервным копиям и обновлениям нужен доступ к папке программы.

## Где хранятся данные

Все данные создаются рядом с программой:

- `guiConfigs` — настройки, профили и подписки;
- `guiBackups` — резервные копии;
- `guiLogs` — журналы;
- `guiTemps` — временные файлы;
- `bin` — ядра, GeoFiles и наборы маршрутизации.

## Чистый релиз

Официальный ZIP не содержит:

- пользовательских профилей;
- подписок;
- статистики;
- журналов;
- резервных копий;
- временных конфигураций;
- результатов сборки.

## Обновление SG Client

1. Отключите активное соединение.
2. Создайте резервную копию в разделе «Обслуживание».
3. Распакуйте новую версию в отдельную папку.
4. Перенесите пользовательские данные только предусмотренным способом.
5. Не смешивайте файлы разных сборок вручную.
