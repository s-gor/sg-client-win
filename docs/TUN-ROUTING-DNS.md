# TUN, Proxy, DNS и маршрутизация

## TUN

TUN создаёт системный сетевой интерфейс и направляет трафик через выбранный профиль.

## Системный прокси

Меняет настройки Windows Proxy после успешного запуска локальных портов. При ошибке прежние настройки должны быть восстановлены.

## Локальный прокси

Запускает HTTP/SOCKS-порты без изменения Windows. Используется для ручной настройки отдельных приложений.

## Умная маршрутизация

- `Direct` — напрямую;
- `VPN` — через профиль;
- `Block` — блокировать.

Основные категории: локальная сеть, российские ресурсы, заблокированные ресурсы, реклама и остальной трафик.

## DNS

SG Client формирует конфигурации отдельно для Xray/sing-box и обычного/TUN-режима. Пока полный JSON не включён, фактическая конфигурация доступна только для чтения.

## Локальная сеть

Разрешает доступ к роутеру, NAS, принтерам и частным подсетям. Не добавляйте широкие исключения без необходимости.
