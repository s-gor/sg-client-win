# Маскировка DPI

Доступны режимы:

- автоматически;
- выключена;
- дробление TLS / ClientHello;
- дробление TLS + шум — экспериментально.

До применения SG Client показывает активные и будущие параметры. После применения результат читается из рабочего `config.json`.

Hysteria 2 работает поверх QUIC/UDP и не получает обычные Xray-настройки TLS fragmentation. AmneziaWG использует параметры маскировки из импортированного профиля.
