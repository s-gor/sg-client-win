# Участие в проекте

Перед Pull Request:

1. Не добавляйте секреты, профили и рабочие журналы.
2. Выполните `dotnet test`.
3. Соберите WPF-проект для `win-x64`.
4. Опишите, какой сценарий SG-Panel или SG-AWG-Panel изменяется.
5. Для UI приложите скриншот до и после.
6. Не добавляйте новые движки и протоколы без согласованной поддержки в панелях SG.
